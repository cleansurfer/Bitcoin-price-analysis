# Bitcoin Price Analysis & Prediction

---> Overview:
- This project performs Exploratory Data Analysis (EDA) and a simple machine
- learning prediction model on Bitcoin historical price data.
- It analyzes historical Bitcoin price data to uncover trends, volatility patterns, and correlations.
- It also prepares a pipeline for forecasting future price movements.
- This demonstrates practical skills in:
    - Time-series analysis
    - Data visualization
    - Statistical modeling
    - Feature engineering
    - Financial data interpretation


---> Features:
- Data Cleaning & Preprocessing:
    - Removing missing values
    - Time index generation
    - Extracting datetime features (day, month, year)
    - Calculating returns & volatility
- Exploratory Time-Series Analysis:
    - Price trends
    - Volume patterns
    - Rolling averages
    - Volatility windows
    - Seasonality inspection
 - Statistical Insights
    - Correlation analysis
    - Moving averages (7d, 30d, 90d)
    - Momentum indicators
- Forecasting Models (optional)
    - This project includes a ready-made pipeline for adding ML models: ARIMA / Prophet / LSTM Neural Networks

 
---> Tech Stack:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels


---> Sample Outputs:
- Bitcoin closing price trend
- Daily volatility chart
- Rolling average comparison
- Return distribution
